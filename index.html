Masternodes

A masternode is an always on server which secures the network by locking up a significant number of coins as collateral to participate in the consensus protocol for a decentralized network. Setting up a masternode is done in several steps. The first of them consists in acquiring a certain quantity of tokens, which is called collateral. Once the collateral of 400,000 SKYR is gathered, a series of more technical steps are to be performed to configure the masternode. Only once the configuration is complete will your masternode start working and generate rewards.
